# YouTube Spam Comments Detection

This project is a machine learning web application that detects spam comments on YouTube videos.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Machine Learning Model
Multinomial Naive Bayes classifier with CountVectorizer for text classification.

## How to Run
1. Install libraries
pip install flask pandas scikit-learn numpy

2. Run the app
python app.py

3. Open browser
http://127.0.0.1:5000
